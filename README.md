# Weather Frontend -- update

Proyecto frontend que muestra información de distintas localidades, su detalle y pronóstico semanal. 
Desarrollado con **HTML5 semántico, Bootstrap 5, CSS/SASS y JavaScript**. 

---

## Características principales

- HTML5 semántico en todas las páginas. 
- Bootstrap 5 para diseño **responsivo mobile-first**. 
- Interfaz renovada siguiendo **metodología BEM** y organización de estilos con **SASS**. 
- Listado de **12+ localidades galácticas** con navegación funcional a página de detalle. 
- **Pronóstico semanal dinámico** por ciudad. 
- Interacción con DOM, tarjetas, eventos y estados. 
- **Efectos hover y glow** en las tarjetas para mejorar la experiencia visual. 
- Layout claro: header, contenido principal y footer. 
- Navegación intuitiva y accesible. 

---

## Estructura del proyecto

/index.html >> Home con tarjetas dinámicas
/detalle.html >> Detalle + pronóstico semanal
/css/styles.css >> Estilos finales con BEM y efectos
/js/app.js >> Lógica principal Home + Detalle
/assets/Backgrounds/fondoGalaxia.jpg >> Imagen de fondo galáctica

> Nota: Se puede usar SASS modularizado y compilar a `styles.css`. Variables, anidamiento y mixins aplicados para mantener consistencia de estilos.

## Cómo ejecutar

1. Clona el repositorio:

git clone https://github.com/motorniko/Update-Clima-Portafolio-M-dulo-3-APLICACIONES-FRONT-END.git

1. Abre index.html en tu navegador.

2. Asegúrate de tener conexión a internet para cargar Bootstrap desde CDN.

3. Haz clic en cualquier tarjeta para navegar a la página de detalle de la ciudad seleccionada.
